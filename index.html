Writing Enonic Guides

Table of Contents
  • What you’ll need

This guide walks you through the process of writing a new guide for inclusion at the Enonic Guides Site. These guides can take different forms—such as short, step-by-step tutorials or longer, more discussion-oriented pieces—but the process for writing and contributing them is the same.

What you’ll need

  • A Git client (command line is fine, or use one of the many excellent GUI clients)

  • A GitHub account

  • A Java Runtime Environment, version 1.8 or better

  • A text editor

  • A basic familiarity with Asciidoc
